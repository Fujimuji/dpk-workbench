# Hax Prime Number Switches

These prime flags describe Hax checkpoint behavior.

- `2`: Rocket Punch disabled
- `3`: Powerblock disabled
- `5`: Seismic Slam disabled
- `7`: Centerless checkpoint on completion
  Project Momentum does not need this.
- `11`: Multilevel level select
  This means map spawn.
- `13`: First checkpoint of a level in a multilevel map
  This means level entrance.
- `17`: Effect lock connection 1
  Project Momentum does not need this.
- `19`: Effect lock connection 2
  Project Momentum does not need this.
- `23`: Effect lock connection 3
  Project Momentum does not need this.
- `29`: Effect checkpoints required
  Project Momentum does not need this.

For current Hax support in this repo, anything after `17` is legacy/obsolete reference material.
